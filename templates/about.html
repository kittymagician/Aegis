About Aegis

Aegis is a web interface to allow non-technical users to run YARA scans on files to classify them.

It was designed by KittyMagician

What is YARA?

YARA is a pattern matching toolkit that is typically used by computer security incident response teams to identify malware.

It supports textural and binary patterns. YARA was designed by Victor Manuel Alvarez for VirusTotal/Chronicle

Technology Stack

Aegis is built with the Flask Microframework, Python3, Yara-Python and Yara

What happens when I upload a sample to Aegis?

The sample is uploaded to the server in the /uploads folder. the yara rules are compiled and sample is then tested againist the super rule output.yar.

Aegis redirects to a results page diplsaying the results and deletes the sample file.

Where can I get rules?

There are many different places you can obtain community yara rules for free. Here's a few exmaple places

  • Yara-Rules
  • Neo23x0 Signature-Base
  • InQuest awesome-yara

If you build some cool rules consider contributing them back to the community!

If you wish to add the rules to the scanner save them to the /rules folder.

Aegis compiles the rules into a single super yara rule called output.yar during each execution (Aegis requires no restarts for changes to be made).

Does Aegis test yara rules?

No. You will need to test the yara rules before including them for use in Aegis.
